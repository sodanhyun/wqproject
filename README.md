# Whisper Question Project
## 은밀한 물음표 [AnonymousQnAService]

> 해당 프로젝트는 강의 중 익명의 이용자가 질문을 전송하고 
> 진행자가 실시간으로 수집된 질문을 확인 및 채택할 수 있으며 
> 관리자가 채택된 질문에 대한 코멘트를 남길 수 있는 서비스를 개발합니다.
